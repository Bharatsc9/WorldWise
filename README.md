# 🌍 WorldWise

WorldWise is a React application that allows users to track and organize the cities they have visited around the world.  
Users can click on the map to add cities, store notes about their trips, and view them in an interactive list.

The application uses an interactive map and a REST API to store and manage visited cities.

## ✨ Features

- 🗺 Interactive world map
- 📍 Add cities by clicking on the map
- 📝 Store notes about each city
- 📋 View visited cities list
- ❌ Delete cities from the list
- 🌐 Data stored through a REST API
- ⚡ Fast and responsive UI
